# products-ecommerce

## Simple Laravel application with livewire, tailwind, echo and pusher
This application demonstrats consuming a 3rd party API to import products.

## Stack
- Laravel
- MySQL (mariadb 10.3)
- soketi 1.4-16 (pusher server)
- products-api (3rd party API)
